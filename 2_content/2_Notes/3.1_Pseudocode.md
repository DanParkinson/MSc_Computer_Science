Ambigiious Insertion Sort:

```
1. Initialise the sort starting from the second element
2. Compare with elements to its left
3. Swap if the current element is smaller
4. Move left until the correct position is found or the start of the array is reached
5. Repeat until the elements are sorted
```

Improved Insertion SOrt

```Python
For i = 1 to length of the list - 1:
    Set the current element as the key(key = list[1]).
    Set J = i -1 (this is the element to the left of the current one).
    While j >= 0 and the element at list[j] is greater than the key:
        swap list[j+1] with list[j].
        Move one position to the left ( j = j-1).
    Insert he ket in its correct position ( list[j +1] = key.)
```
